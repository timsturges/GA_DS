Data Visualization With Pandas, Matplotlib, & Seaborn¶

The major chart types used in graphing data with Python.

  • Matplotlib: The major library that others are built off of
  • Pandas: Has built in matplotlib functions that allow you to graph based off of a dataframe
  • Seaborn: User-friendly version of matplotlib, has a number of custom chart types that are very handy.

A really important detail is getting the correct dataframe for each graph!
